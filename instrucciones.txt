INSTRUCCIONES MÓDULO IPS

1. Instalar dependencias (desde la carpeta raíz):
   pip install scapy pandas numpy scikit-learn joblib

2. Entrenar modelo (desde la carpeta raíz):
   python IPS/entrenarModelo.py

3. Ejecutar captura de red (desde la carpeta raíz):
   python IPS/capturaRed.py
   
   NOTA: Para que el bloqueo automático funcione, ejecuta como Administrador.
   El sistema bloqueará automáticamente las IPs de atacantes detectados.

4. Simular ataques (opcional, para pruebas):
   python IPS/simularAtaques.py
   python IPS/simularAtaques.py "Web Attack - XSS"  (ataque específico)

5. Gestionar bloqueos:
   python IPS/gestionarBloqueos.py
   - Ver IPs bloqueadas
   - Desbloquear IPs
   - Bloquear IPs manualmente
   - Limpiar bloqueos expirados
